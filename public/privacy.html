개인정보처리방침

이 서비스(시간표 생성기)는 이용자의 개인정보를 서버로 수집·전송·저장하지 않습니다. 업로드한 시수표(교사 이름 포함)를 비롯한 모든 데이터는 이용자의 브라우저 안에서만 처리됩니다.

1. 처리하는 데이터와 처리 위치

  • 시수표 엑셀 파일(교사 이름·과목·학급 등)과 설정은 이용자 컴퓨터의 브라우저 메모리에서만 처리됩니다.
  • 시간표 계산(솔버)도 전부 브라우저 안(WebAssembly)에서 실행되며, 어떤 데이터도 운영자 또는 제3자 서버로 전송되지 않습니다.
  • 내려받는 결과 파일(엑셀·이어돌리기 JSON)은 이용자 컴퓨터에만 저장됩니다. 브라우저 탭을 닫으면 작업 중 데이터는 사라집니다.
  • 쿠키·로그인·광고·행동 추적 도구를 사용하지 않습니다.
  • 방문자 수 집계 — 오늘·누적 방문 횟수라는 숫자 두 개만 세며, IP·기기정보 등 개인을 식별할 수 있는 정보는 수집·저장하지 않습니다. 하루 한 번만 세기 위한 표시(마지막 방문 날짜)는 이용자 브라우저(localStorage)에만 남습니다.

2. 외부 서비스

  • 호스팅(GitHub Pages) — 이 사이트는 GitHub Pages에서 제공됩니다. 접속 과정에서 IP 주소 등 기본 접속 기록이 GitHub의 정책에 따라 처리될 수 있습니다. GitHub 개인정보처리방침
  • 규칙 추가 요청 게시판(GitHub Issues) — 게시판 글·댓글은 GitHub Issues 공개 저장소에 저장됩니다. 작성 시 이용자의 GitHub 계정 이름이 공개되며, 이는 GitHub의 정책을 따릅니다. 글을 읽기만 할 때는 계정이 필요 없습니다.

3. 이용자의 권리

운영자는 이용자의 개인정보를 보유하지 않으므로 별도의 열람·정정·삭제 절차가 없습니다. 게시판에 남긴 글·댓글의 수정·삭제는 GitHub에서 본인 계정으로 직접 할 수 있습니다.
